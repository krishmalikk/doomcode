Features

Mobile Control

Control your AI coding agent directly from your smartphone. Give commands, review outputs, and stay connected wherever you are.

E2E Encryption

All communication between your phone and desktop is end-to-end encrypted. Your code and commands stay private.

Diff Review

Review code changes proposed by the AI before they're applied. Approve or reject patches with a single tap.

Voice Input

Use speech-to-text for hands-free commands. Perfect for when you're away from your keyboard.

QR Pairing

Connect your phone to your desktop in seconds. Just scan the QR code and you're ready to go.

GitHub Integration

Create pull requests directly from your phone. Share GitHub tokens securely with your desktop session.
