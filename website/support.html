Getting Started

New to DoomCode? Here's how to get set up:

  1. Install the Desktop CLI: Run npm install -g doomcode to install the desktop component.
  2. Download the Mobile App: Get DoomCode from the App Store (iOS) or Google Play (Android).
  3. Start a Session: Run doomcode start in your terminal. A QR code will appear.
  4. Pair Your Device: Open the mobile app and scan the QR code to connect.
  5. Start Coding: Send commands from your phone and watch your AI agent work!

System Requirements

Desktop

  • Node.js 18 or later
  • macOS, Windows, or Linux
  • One of the supported AI agents installed (Claude, Codex, or Gemini CLI)

Mobile

  • iOS 12.0 or later
  • Android 8.0 or later
  • Internet connection (Wi-Fi or cellular)

Troubleshooting

Can't connect to desktop

Make sure both devices are connected to the internet. The connection uses relay servers, so they don't need to be on the same network. Try restarting the desktop session with doomcode start.

QR code not scanning

Ensure your camera has permission to scan QR codes. Try adjusting the lighting or moving closer to the screen. You can also manually enter the session code if available.

Agent not responding

Check that your AI agent CLI (Claude, Codex, or Gemini) is properly installed and configured on your desktop. The agent must be accessible from the command line.

Connection drops frequently

This may be due to unstable network conditions. Try connecting to a more stable network. DoomCode will automatically attempt to reconnect when the connection is lost.
